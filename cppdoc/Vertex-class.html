class Vertex

Each object of this class represents a single vertex (or node) in a graph. The vertex object itself stores some useful data fields that can be used in various graph algorithms, like path-searching algorithms such as Dijkstra's algorithm.

See also: BasicGraph, Edge

Available since: 2014/02/01 version of C++ library

Method detail

int getColor();
Returns the color of the given vertex, as set previously by a call to setColor. If setColor has not been called previously, the return value is unspecified.

Usage:

int color = v.getColor();

void resetData();
Sets the cost, previous, and visited fields and color back to their initial values of 0, NULL, and false respectively.

Usage:

v.resetData();

void setColor(int color);
Sets the color of the given vertex to the given color. Future calls to getColor will return this color.

Usage:

v.setColor(color);

string toString() const;
Returns a string representation of this vertex, such as "Vertex{name=r13c42, cost=11, visited=true, previous=r12c41, neighbors={r12c41, r12c43}}".

Usage:

string str = v.toString();
