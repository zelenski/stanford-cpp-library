◆ initializeLibraryStudentThread()

void initializeLibraryStudentThread ( )

This is for any initialization that needs to be done in the student's thread rather than on the Qt GUI main thread.

Currently this is used primarily to set up exception handlers for the student's thread so we can print usable stack traces.
