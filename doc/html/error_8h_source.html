31  * This exception is thrown by calls to the <code>error</code>
32  * function. Typical code for catching errors looks like this:
33  *
34  *<pre>
35  * try {
36  * ... code in which an error might occur ...
37  * } catch (ErrorException& ex) {

42  * If an <code>ErrorException</code> is thrown at any point in the
