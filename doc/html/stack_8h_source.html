44  * Class: Stack<ValueType>
45  * -----------------------
46  * This class models a linear structure called a <b><i>stack</i></b>
47  * in which values are added and removed only from one end.
48  * This discipline gives rise to a last-in/first-out behavior (LIFO)
49  * that is the defining feature of stacks. The fundamental stack
50  * operations are <code>push</code> (add to top) and <code>pop</code>
51  * (remove from top).
52  */
